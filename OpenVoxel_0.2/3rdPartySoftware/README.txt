
QT 4 (open source license)

You will need the QT4 sdk or the QT4 sources appropriate for your compiler and OS

Windows:
MSVC 10: Etiher the SDK or the Sources will work
MSVC 11: Untill released you will have to follow the QT docs to rebuild the QT libs under msvc11

Linux:
install qt4-dev package.  QT may complaing if you don't have the phonon backend installed - just follow their directions if this happens.

Mac: (untested)

Install the qt4-dev package.
